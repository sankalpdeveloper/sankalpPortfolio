Fire Alarm Sensor with IOT

In this completed project, I successfully designed and built an IoT-enabled fire alarm sensor, integrating hardware components and software development. The sensor effectively detects the presence of smoke or fire and sends real-time alerts to a web-based dashboard and a mobile application, enabling remote monitoring and control. This project has significantly enhanced fire safety by providing timely notifications and enabling efficient remote management.

Functionalities:

  • The core functionality of the system is to detect the presence of fire or smoke. It uses specialized smoke detectors and/or heat sensors for this purpose.
  • When the sensor detects smoke or fire, it immediately triggers an alert. This alert is sent in real-time to a central monitoring system.
